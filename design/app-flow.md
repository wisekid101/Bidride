# App Flow

> **Status: PLACEHOLDER — Wireframes to be created in Phase 2**

## Rider App — Core Flow
1. Open app → Enter destination
2. See fare estimate
3. Choose: **Accept Standard Fare** or **Submit a Bid**
4. If bid: enter bid amount → drivers notified
5. Driver accepts / counters / rejects
6. Match confirmed → driver en route
7. Trip in progress → arrival → payment → rating

## Driver App — Core Flow
1. Go online → enter available status
2. Receive ride request (standard) or bid offer
3. Review: pickup location, destination, fare/bid
4. Accept / Reject / Counter (on bids)
5. Navigate to rider
6. Trip in progress → drop off → payment → rating

## Admin Portal — Core Flow
1. Dashboard: active rides, drivers online, revenue
2. Driver management: approvals, background check status
3. Rider management: accounts, disputes
4. Pricing controls: base rates, bid floors, multipliers
5. Incident management
